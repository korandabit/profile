# backend of markkoranda.com

You are viewing the repo for [my webpage](http://markkoranda.com).

## Structure

- `_posts/` - Blog posts in markdown (2012-2024)
- `docs/` - White papers, CV, and maintenance documentation
- `tools/` - Maintenance scripts for blog health checks
- `images/` - Site images and assets
- `photos/` - Photo gallery content

## Maintenance

Blog maintenance scripts and reports are in [`tools/`](tools/) and [`docs/maintenance/`](docs/maintenance/).
